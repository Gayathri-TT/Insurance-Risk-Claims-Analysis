# Insurance-Risk-Claims-Analysis

## 1. Project Overview 
This project focuses on building a centralized Power BI dashboard for an insurance company to better understand its policyholder base and claim patterns. Currently, policy and claims data are scattered across multiple sources, making it difficult for stakeholders to track performance and identify trends.

## 2. Project Objectives 
To centralize scattered policy and claims data into a single Power BI dashboard.
To provide clear visibility into key insurance KPIs for decision-making.
To analyze customer, vehicle, and claim patterns through interactive visuals.
To help stakeholders identify trends, risk factors, and performance insights.

## 3. Tech Stack
The dashboard was built using to build the dashboard.
* Power BI Desktop - Main data visualization platform used for report creation.
* Power query - Data transformation and cleaning layer for reshaping and praparing the data
* DAX- Used for calculated measures.
* File Foramt - .pbix for development and .png for dashboard previews.

## 4. Key Insights Delivered
The dashboard helps stakeholders understand
* Policy distribution across genders  
* Claim amount trends  
* Total claim amount by education & marital status  
* Claim frequency patterns  
* Total policies and cliam by coverage zone.
